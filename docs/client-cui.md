# Client.java のCUI修正内容

## 概要

`apps/Client.java` において、サーバと接続した後もクライアントが起動し続け、CUIから操作できるように修正した。

## 具体的な内容

`inputLoop()` を追加することで、標準入力からCUIコマンドを受け付けられるようにした。

現在対応しているコマンドは以下の通りである。

| コマンド          | 内容                                                                                         |
| ----------------- | -------------------------------------------------------------------------------------------- |
| `answer <number>` | この段階では、クイズを選択肢の番号で回答することを想定している。ただし、後で修正可能である。 |
| `disconnect`      | サーバに対して切断要求を送る。                                                               |
| `quit` / `exit`   | `disconnect` と同様の機能である。とりあえず追加したものであるため、後で消してもよい。        |
| `help`            | 操作方法を示す。                                                                             |

`receiveLoop()` を追加することで、サーバからのメッセージを継続的に受信している。

`receiverThread` は、`receiveLoop(in)` を別スレッドで動かすためのものである。

`printHelp()` により、使用できるコマンドをユーザに明示している。
