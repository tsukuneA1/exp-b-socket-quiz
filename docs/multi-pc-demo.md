# 複数PCデモ手順

最終発表で複数PCから同じクイズ部屋に参加するための手順。

## 目的

サーバPCを1台用意し、他のPCからGUIまたはCUIクライアントで参加する。

早押し確認では、複数PCで同時に回答ボタンを押し、正解者が必ず1人だけになることを確認する。

## 前提

- 全PCが同じWi-Fiまたは同じLANに接続されていること
- 全PCでJava 21が使えること
- サーバPCのファイアウォールが使用ポートを遮断していないこと
- デフォルトポートは `8080`

## サーバPCで行うこと

サーバPCのIPアドレスを確認する。

macOSの場合:

```bash
ipconfig getifaddr en0
```

有線LANの場合や上記で表示されない場合:

```bash
ifconfig
```

例として、サーバPCのIPアドレスが `192.168.1.10` だったとする。

サーバを起動する。

```bash
make server
```

ポートを指定する場合:

```bash
make server-port port=9090
```

## クライアントPCで行うこと

GUIで参加する場合:

```bash
make gui-client-host name=Alice host=192.168.1.10 port=8080
```

CUIで参加する場合:

```bash
make client-host name=Alice host=192.168.1.10 port=8080
```

ポートを9090にした場合:

```bash
make gui-client-host name=Alice host=192.168.1.10 port=9090
```

直接Mavenで起動する場合:

```bash
mvn exec:java -Dexec.mainClass=GuiClient -Dexec.args="Alice 192.168.1.10 8080"
```

## 標準入力・操作方法

GUIの場合:

- `Ready` ボタンを押して準備完了にする
- 出題後、4つの選択肢ボタンから回答する
- 終了後、`もう一度対戦する` または `部屋を抜ける` を押す

CUIの場合:

- `ready` を入力して準備完了にする
- 出題後、選択肢番号を数字で入力する
  - 例: `2`
- 切断する場合は `d`、`quit`、`exit` のいずれかを入力する

## 早押し確認手順

1. サーバPCで `make server` を実行する
2. 2台以上のクライアントPCから `make gui-client-host ...` で参加する
3. 全員が `Ready` を押す
4. 問題が表示されたら、複数人が同時に正解ボタンを押す
5. `ROUND_END` の表示またはGUIログで、正解者が1人だけであることを確認する
6. スコアが正解者1人にだけ加算されることを確認する

## 接続できない場合の確認

- サーバPCとクライアントPCが同じネットワークにいるか
- クライアント側の `host` にサーバPCのIPアドレスを指定しているか
- サーバが起動しているか
- ポート番号がサーバとクライアントで一致しているか
- サーバPCのファイアウォールでポートが遮断されていないか
